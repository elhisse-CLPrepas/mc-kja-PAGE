## Points a valider avant publication

- Scanner les QR codes avec un smartphone.
- Tester les liens externes depuis le navigateur.
- Verifier que les mentions `coach certifiee`, `naturopathe holistique` et `formations certifiantes` sont autorisees pour diffusion publique.
- Relire les textes pour confirmer l'absence de promesse medicale.
- Tester le rendu mobile, tablette et desktop.

## Tests recommandes

1. Ouvrir `index.html` localement.
2. Tester le menu mobile.
3. Cliquer sur les boutons du hero.
4. Parcourir les affiches QR.
5. Tester les boutons `Ouvrir le lien`.
6. Tester les boutons `Copier le lien`.
7. Verifier le bouton retour en haut.
